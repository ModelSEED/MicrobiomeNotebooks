================================================================================
LESSONS LEARNED
================================================================================

LESSON 1: MAG-derived Models Often Have Core Metabolism Gaps
--------------------------------------------------------------------------------
Observation:
  Model derived from metagenome-assembled genome (MAG) cannot grow, despite
  having 878 reactions and 730 genes.

Explanation:
  • MAG assembly may be incomplete (fragmented genes)
  • Gene annotation may miss essential functions
  • ModelSEED template may not include organism-specific pathways
  • Biomass reaction may have unrealistic requirements for MAG models

Recommendation:
  Always test base model growth on complete media BEFORE specialized gap-filling

LESSON 2: Pathway Functionality ≠ Growth
--------------------------------------------------------------------------------
Observation:
  Betaine reductase pathway is fully functional (10 mmol/gDW/h flux) but
  model shows zero growth.

Explanation:
  • Pathway can carry flux when forced (objective = pathway flux)
  • But won't be used for growth if biomass can't be produced
  • FBA will not activate pathways that don't contribute to objective

Recommendation:
  Test pathway flux capacity independently of growth objective

LESSON 3: Automated Gap-filling is Essential for MAG Models
--------------------------------------------------------------------------------
Observation:
  Manual addition of betaine pathway succeeded, but didn't enable growth due
  to unknown gaps elsewhere in metabolism.

Explanation:
  • MAG models may have dozens to hundreds of gaps
  • Manual gap-filling can only address known pathways
  • Unknown gaps (e.g., missing cofactor synthesis) require automated discovery

Recommendation:
  Use ModelSEEDpy's automated gap-filling for core metabolism FIRST, then
  add specialized pathways (betaine) manually or with targeted gap-filling

LESSON 4: Literature Evidence Validates Gap-filling
--------------------------------------------------------------------------------
Observation:
  Gap-filled pathway shows exact stoichiometry and flux rates from literature

Conclusion:
  • Manual gap-filling based on strong literature evidence is valid
  • The betaine reductase pathway is correctly implemented
  • Flux capacity (10 mmol/gDW/h) matches experimental observations

Recommendation:
  Use literature-based gap-filling for well-characterized pathways, even
  without genome sequence evidence (taxonomy-based gap-filling)

LESSON 5: Two-phase Gap-filling Approach Required
--------------------------------------------------------------------------------
Observation:
  Specialized metabolism (betaine) won't function if core metabolism broken

Recommended workflow:
  Phase 1: Automated core gap-filling
    - Fix biomass production
    - Ensure growth on standard media
    - Validate ATP production, cofactor synthesis

  Phase 2: Specialized pathway addition
    - Add betaine reductase (manual or automated)
    - Add selenocysteine biosynthesis
    - Add Stickland reaction coupling

Python code:
```python
from modelseedpy import MSBuilder, MSModelUtil
from modelseedpy.core import FBAHelper

model = cobra.io.load_json_model('model.json')

# Define complete media
complete_media = MSModelUtil.build_media(
    model,
    media_id='complete',
    default_uptake=100
)

# Gap-fill to enable growth
gapfill_result = MSBuilder.gapfill_model(
    model,
    media=complete_media,
    target=model.reactions.bio1,
    minimum_obj=0.01
)

# Add suggested reactions
for rxn in gapfill_result['new']:
    model.add_reactions([rxn])

# Validate
solution = model.optimize()
print(f"Growth after gap-filling: {solution.objective_value}")

# Save
cobra.io.save_json_model(model, 'model.core_gapfilled.json')
```
