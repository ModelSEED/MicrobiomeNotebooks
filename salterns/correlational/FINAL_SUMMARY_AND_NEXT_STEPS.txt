================================================================================
HYPOTHESIS TESTING FRAMEWORK
================================================================================

## Media Conditions Defined:

1. **Betaine Minimal**
   - Tests: Can organisms grow on betaine as sole carbon source?
   - Expected positive: Halanaerobiales, Peptostreptococcales
   - Components: Betaine + minerals + trace metals

2. **Betaine + Selenite**
   - Tests: Does selenocysteine enhance betaine metabolism?
   - Expected positive: Peptostreptococcales (documented requirement)
   - Components: Betaine minimal + selenite

3. **Betaine + Amino Acids (Stickland)**
   - Tests: Coupled betaine reduction + amino acid oxidation?
   - Expected positive: Peptostreptococcales (HIGH confidence)
   - Components: Betaine + selenite + alanine, leucine, valine, histidine

4. **Amino Acids Only**
   - Tests: Amino acid fermentation capability?
   - Expected positive: Synergistales (VERY HIGH confidence)
   - Components: 12 amino acids + minerals

## Key Measurements:

For each test, the framework measures:
- Growth rate (biomass or ATP maintenance)
- Betaine uptake flux (mmol/gDW/hr)
- TMA production flux (expected 1:1 with betaine)
- Acetate production flux (expected 1:1 with betaine)
- Amino acid uptake (for Stickland tests)
- Presence of betaine reductase (rxs17220_c0)
- Presence of TMA transport

## Validation Logic:

Each result is compared to literature-based expectations:
- ✓ PASS: Model behavior matches experimental evidence
- ✗ FAIL: Discrepancy identified with specific description

Discrepancies are categorized and tallied by:
- Taxonomic order
- Test condition
- Discrepancy type
